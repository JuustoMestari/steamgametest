How to use this webApp ?

  1. Select a game in the list. You can randomize the list by clicking the Randomize button
  2. When a game is selected, its info are displayed
  3. The most popular (=most view count) 3 youtube videos for this game are retrieved
  4. The most popular live twitch stream is retrieved
  5. 4 screenshots from the game are shown

How does this work?

  1. The best 100 games from steam (last 2 weeks) are retrieved from steamspy
  2. 10 games are randomly selected
  3. When you click randomize, the list is filled with 10 new random games

What components are used for this WebApp?

  • AngularJS
  • Webpack
  • Bootstrap
  • Python with flask for the backend
  • Youtube API
